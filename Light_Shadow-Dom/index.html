Sample1

  • Light DOM の class1, class2 が適用される
  • Shadow DOM の class3, class4 が適用されない
  • script はページ全体に適用される

Sample2: Light DOM に template で読み込み

  • Light DOM の class1, class2 が適用される
  • Shadow DOM の class3, class4 が適用されない
  • script はページ全体に適用される

Sample3: Shadow DOM に template で読み込み

  • Light DOM の class1, class2 が適用されない
  • Shadow DOM の class3 が適用される
  • Shadow DOM の class4 が適用されない
  • script はページ全体に適用される

Sample4: Shadow DOM にベタ打ちで読み込み

  • Light DOM の class1, class2 が適用されない
  • Shadow DOM の class3 が適用されない
  • Shadow DOM の class4 が適用される
  • script が読み込まれない

Sample5: iframe で読み込み

  • iframe 内の css/js が適用される
  • iframe 外の css/js が適用されない

結論

  • Light DOM で読み込んだ css は、 Light DOM 内にのみ適用される
  • Shadow DOM で読み込んだ css は、その Shadow DOM 内にのみ適用される
  • template タグで読み込んだ js は、 Light/Shadow DOM に関わらず、ページ全体に適用される
  • innerHTML では、 js を読み込めない
  • iframe は、内外に影響し合わない
